## Coindump
A Flutter app that retrieves the name, symbol and price from [CoinCap API 2.0](https://docs.coincap.io/#89deffa0-ab03-4e0a-8d92-637a857d2c91) and displays them in a list view.

# Dependencies
- web_socket_channel
- http
